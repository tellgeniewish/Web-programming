html이 태그 기반 언어의 표준은 아님

XML

<table> // <>안의 table은 태그 이름
   <tr>
      <td>태그 바디(텍스트, 또 다른 태그 들어갈 수 있음)</td>
   </tr>
</table>
=에 붙어 나오는건 (태그)속성, attribute

액션 태그 종류

시작한 후에 끝나는 코드 없으면 오류남
<jsp:태그키워드 태그속성="태그값">
</jsp:태그키워드>// 반드시 끝나는 태그 있어야 함

<jsp:include에는 태그 바디 없을 수도 있음
있다면 <jsp:param 이 올 수 있음
<jsp:include page="sub.jsp" /> // 태그 바디 없으면 한 문장 내에서 끝나는 태그 />로 끝내줘야 함

-소스의 삽입
-지역변수 중복 선언의 문제가 발생할 수 있음

액션 태그 include(실행 결과가 태그 자리에 들어감)
-결과의 삽입
